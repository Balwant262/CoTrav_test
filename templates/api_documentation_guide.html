CoTrav – Public API’s

Introduction

We are based at Head office in Delhi & Branch offices in Pune & Gurgaon and bring to you all travel related services under one roof. Our main line of operations is to serve the B2B agents and corporate clients with the best possible travel related services all across India. We are handling approximately 100 corporates at this time all across India. Handled by a team of Professionals with a 24X7 support call center consisting of experienced staff of the travel industry.

Product Overview
  • Cotrav is the booking system for Complete Travel Management for Corporate at one place.
  • The platform manages all forms of travel viz: flights, hotels, buses, trains, Cabs etc.
  • The platform offers end-to-end solutions – from pre-trip processes like approval to making bookings to compliance and analytics and finally post-travel reconciliation and expense management – for the organization.
  • All the bookings that are made in the system are compliant with the policies of every client.
Technology Overview

Cotrav Booking System is developed on a very secure Django Framework with database as MySQL. Django is a premier Python Framework with over 75 million+ downloads.

Some of the salient features of Django Framework are as follows:

  • Pure object oriented web application framework.
  • Advanced MVC implementation.
  • Supports multi databases including MySQL, PostgresSQL, SQLite etc.,
  • Session management.
  • Data Encryption.
  • Flexible URI Routing.
  • Provides RESTful API development support.
  • Code reusable and easier to maintain.
  • Supports strong cryptography tools and password hashing techniques.
  • All the transactions done through a secure connection.

API Overview

Integrate your Applications with our API’s to raise the booking requests with us.

The Stripe API is organized around REST. Our API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

Authentication

To get the Authentication key using the credentials provided by CoTrav

URL : https://cotrav.co/api/get_corporate_auth_token
Content-Type : application/json
Method : POST
Request 
{
    "corporate_id":"####ID#####",
    "user_id":"####USERNAME#####",
    "user_password":"####PASSWORD#####"
}
Response Type : JSON
Response 
{
    "success": 1,
    "message": "user details match successfully",
    "access_token": "########ACCESS_TOKEN############"
}
